# Обновлённые инструкции по деплою в Google Cloud

## Быстрая установка

1. **Создайте VM в Google Cloud:**
   - Ubuntu 22.04 LTS
   - e2-micro или e2-small
   - Включите HTTP/HTTPS в firewall

2. **Подключитесь по SSH и выполните:**
```bash
# Клонируйте репозиторий
git clone https://github.com/BitaliyS/case_status_look_bot_v2.git
cd case_status_look_bot_v2

# Сделайте скрипт исполняемым и запустите
chmod +x setup_cloud.sh
./setup_cloud.sh
```

3. **Настройте переменные окружения:**
```bash
# Создайте .env файл
nano .env
```

Добавьте в .env:
```
BOT_TOKEN=ваш_токен_бота
ADMIN_CHAT_ID=ваш_chat_id
RUCAPTCHA_API_KEY=ваш_ключ_капчи
PROXY_HOST=хост_прокси
PROXY_PORT=порт_прокси
PROXY_USER=пользователь_прокси
PROXY_PASS=пароль_прокси
```

4. **Загрузите сервисный JSON файл:**
```bash
# Загрузите case-look-b1a1e3beddf1.json в корень проекта
```

5. **Настройте автозапуск:**
```bash
# Скопируйте сервис
sudo cp case-bot.service /etc/systemd/system/

# Включите и запустите сервис
sudo systemctl daemon-reload
sudo systemctl enable case-bot
sudo systemctl start case-bot

# Проверьте статус
sudo systemctl status case-bot
```

## Проверка работы

```bash
# Посмотреть логи
sudo journalctl -u case-bot -f

# Перезапустить сервис
sudo systemctl restart case-bot

# Остановить сервис
sudo systemctl stop case-bot
```

## Возможные проблемы

1. **Chrome не запускается:** Проверьте, что установлен Chrome и Chromedriver
2. **Ошибки импорта:** Убедитесь, что все зависимости установлены
3. **Проблемы с правами:** Запускайте от root или настройте права для пользователя
